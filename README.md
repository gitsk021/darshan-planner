A modern AI-powered web app to help pilgrims plan their spiritual journeys seamlessly.

## ✨ Features
*   **AI Itinerary Generation**: Custom tour plans based on your starting city.
*   **Temple Directory**: Detailed information on major Darshan sites.
*   **Responsive Design**: Built with Tailwind CSS for mobile and desktop.

*   Internet Computer (ICP) - Blockchain Hosting
*   Tailwind CSS - Styling

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com`
2. Open `index.html` in your browser.
